# Curly

Small Android application to perform GET requests and display output of result.

Built in endpoints

- Shodan's InternetDB
- Mullvad's "Am I Mullvad"
- ip-api
- wttr.in
- more as I find them

Custom endpoint (just type a URL in)

- Example: https://api.airplanes.live/v2/mil
